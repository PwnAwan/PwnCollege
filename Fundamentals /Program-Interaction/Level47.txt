Read the challenge file by typing /challenge/embryoio_level47
The challenge checks for a specific parent process : shellscript
The challenge checks for a specific process at the other end of stdin : rev
The challenge will check for a hardcoded password over stdin : icgtasos

Write the script as:
#!/bin/bash
rev | /challenge/embryoio_level47
Run the script as "bash my_script.sh"  and then enter the password in reverse as "sosatgci".
But it will not show the flag. Just enter ctrl+d to view the flag.
